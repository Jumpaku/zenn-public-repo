# Zenn 公開リポジトリ

PADAone(estra) 名義で [Zenn](https://zenn.dev/estra) に無料公開している記事や本を管理しているリポジトリです。

タイポや日本語などの修正などをプルリクエストで受け付けています。

有料公開しているもの(あるいはその予定のもの)は別の Private リポジトリで管理しているので注意してください。
